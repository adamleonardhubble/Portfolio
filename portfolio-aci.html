Synopsis

A network intrusion detection system developed from a neural network model, that is capable of differentiating between bad connections, namely intrusions or attacks, and good, ordinary connections.

The network intrusion detection system supports both binary and categorical modes of network connection classification, for the publicly available KDD Cup 1999 dataset.
